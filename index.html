El estudio

Minimalismo visual, control tecnico y una estetica sobria.

El sitio fue replanteado para verse como un estudio fotografico profesional: mas silencioso, mas serio y con la imagen como protagonista. La interfaz reduce adornos y deja que el trabajo hable con claridad.

Espacio de trabajo

Un estudio preparado para retrato, producto y contenido comercial.

Luz

Control de iluminacion para piel, textura y volumen

Esquemas ajustados para retratos profesionales, editoriales discretas y fotografia de producto con acabado limpio.

Proceso

Produccion ordenada desde la idea hasta la entrega

Cada sesion se trabaja con planeacion, referencias visuales, direccion en set y una seleccion final pensada para uso comercial y digital.

Entrega

Archivos listos para web, campañas y materiales impresos

Imagenes editadas con consistencia, pensadas para catálogos, redes, branding personal o presentaciones corporativas.
